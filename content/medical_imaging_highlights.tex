\section{Medical Imaging \& Viewer Development Highlights}
\begin{itemize}
  \item Built a complete DICOM medical imaging viewer using Cornerstone.js, serving as the leading medical image viewer in Egypt and the MENA region
  \item 3 merged PRs in Cornerstone 3D (ultrasound metadata, ReferenceLines fixes, loop scroll tools), contributing to core imaging library improvements
  \item Implemented comprehensive viewport tools: windowing (WL/WW), pan, zoom, stack scroll, and cine playback
  \item Optimized image loading pipelines for large DICOM studies, handling multi-frame images and series navigation
  \item Developed annotation workflows and ROI tools for radiologists, integrated with reporting systems
  \item Collaborated with regulatory teams on FDA clearance documentation and validation for platform components
  \item Experience with DICOM parsing, metadata extraction, pixel data handling, and clinical workflow integration
\end{itemize}
